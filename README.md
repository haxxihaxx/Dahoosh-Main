# داهوش
### پلتفرم هوشمند تصحیح آزمون و آموزش با هوش مصنوعی

---

## معرفی محصول

داهوش یک نرم‌افزار دسکتاپ بومی است که چهار ابزار هوش مصنوعی قدرتمند را در قالب یک رابط کاربری یکپارچه و شیک برای معلمان، دانش‌آموزان و نهادهای آموزشی ایرانی گردآورده است. این برنامه با معماری Electron و پشتوانه Flask، به‌صورت کامل آفلاین روی سیستم کاربر اجرا می‌شود و برای پشتیبانی از زبان فارسی و محتوای دو‌زبانه از پایه طراحی شده است.

داهوش ساعت‌هایی که معلمان صرف تصحیح دستی برگه می‌کنند را به چند ثانیه کاهش می‌دهد، اسناد آموزشی را به پادکست صوتی تبدیل می‌کند، سوالات ریاضی را گام‌به‌گام حل می‌کند و به دانش‌آموزان با یک دستیار چت هوشمند پاسخ می‌دهد. همه این قابلیت‌ها در یک پنجره، بدون نیاز به مرورگر و بدون نصب‌های پیچیده.

---

## مشکلی که حل می‌کنیم

سیستم آموزشی ایران با چالش‌هایی روبروست که هر معلم و دانش‌آموزی آن‌ها را به‌خوبی می‌شناسد:

**برای معلم:** تصحیح دستی ده‌ها تا صدها ورقه امتحانی، هر بار با صرف ساعت‌ها وقت، با خطر خستگی و بی‌دقتی، و بدون بازخورد سازنده برای دانش‌آموز.

**برای دانش‌آموز:** دسترسی محدود به توضیح مفاهیم درسی در ساعاتی که معلم در دسترس نیست، و نبود ابزاری که بتواند مسئله ریاضی را از روی عکس حل کند یا محتوای کتاب درسی را ساده توضیح دهد.

**برای هر دو:** ابزارهای موجود یا به‌درستی از فارسی پشتیبانی نمی‌کنند، یا نیاز به اتصال دائم به اینترنت دارند، یا رابط کاربری آن‌ها برای کاربر عادی پیچیده است.

داهوش این سه شکاف را با یک نرم‌افزار پل می‌زند.

## قابلیت‌های اصلی

### تصحیح هوشمند آزمون

این ماژول گردش‌کار تصحیح آزمون را از ابتدا تا انتها خودکار می‌کند.

معلم فایل PDF کلید پاسخ و برگه دانش‌آموز را بارگذاری می‌کند. سیستم با یک فراخوانی به Vision API متن دست‌نوشته یا تایپ‌شده را استخراج می‌کند، آن را در برابر کلید پاسخ ارزیابی می‌کند و یک گزارش PDF حرفه‌ای با جزئیات کامل تولید می‌کند. گزارش نهایی شامل نمره هر سوال، نقاط قوت، نقاط قابل بهبود و بازخورد فارسی است.

موتور تصحیح از دو الگو پشتیبانی می‌کند: آزمون‌های تستی با نمره‌دهی صفر و صد، و سوالات تشریحی با ارزیابی وزن‌دار بر اساس صحت محتوا، کامل بودن پاسخ، انسجام ادبی و مثال‌های پشتیبان. تمام این فرآیند برای متن فارسی، انگلیسی و دو‌زبانه بهینه شده است.

### دستیار RAG روی اسناد شما

این ماژول به کاربر اجازه می‌دهد هر سند متنی یا PDF را آپلود کند و با زبان طبیعی از آن بپرسد.

در پشت صحنه، سیستم سند را به قطعه‌های همپوشان تقسیم می‌کند، برداری TF-IDF برای هر قطعه محاسبه می‌کند و با شباهت کسینوسی مرتبط‌ترین بخش‌ها را پیدا می‌کند. سپس این بخش‌ها به GPT-4o داده می‌شوند تا پاسخی دقیق و مستند تولید کند. کل این چرخه جستجوی برداری بدون پایگاه داده خارجی و به‌صورت درون‌برنامه‌ای اجرا می‌شود، که داهوش را از رقبا متمایز می‌کند.

### پادکست AI از هر سند

کافی است یک فایل متنی یا PDF بارگذاری شود. داهوش ابتدا یک فیلمنامه گفتگوی طبیعی با دو مجری (Alex و Sara) می‌نویسد که ایده‌های اصلی سند را با لحن پادکستی پوشش می‌دهند، سپس با Text-to-Speech آن را به فایل صوتی MP3 تبدیل می‌کند. خروجی قابل دانلود است. این قابلیت برای معلمانی که می‌خواهند محتوای درسی را در قالب صوتی برای دانش‌آموزان تهیه کنند استفاده‌های فراوانی دارد.

### حل‌کننده هوشمند ریاضی

دانش‌آموز می‌تواند مسئله را تایپ کند یا عکس آن را بارگذاری کند. سیستم با GPT-4o Vision مسئله را شناسایی، دسته‌بندی (جبر، هندسه، حساب دیفرانسیل و انتگرال و ...) و گام‌به‌گام حل می‌کند. خروجی به‌صورت JSON ساختاریافته با عنوان هر گام، توضیح، فرمول و خلاصه نهایی ارائه می‌شود. پاسخ‌ها در صورت فارسی بودن سوال به فارسی و در صورت انگلیسی بودن به انگلیسی نوشته می‌شوند.

## معماری فنی

داهوش از یک معماری دو‌لایه استفاده می‌کند که به‌طور هوشمندانه‌ای فناوری‌های وب و بومی را ترکیب می‌کند.

```
داهوش/
├── main.js                پردازش اصلی Electron - مدیریت پنجره و راه‌اندازی Flask
├── preload.js             پل IPC امن با contextIsolation
├── app.py                 بک‌اند Flask - نقطه ورود API
├── handlers/
│   ├── flask_api_handler.py       مسیرهای API اصلی (تصحیح، گزارش)
│   └── new_features_handler.py    مسیرهای RAG، پادکست، ریاضی، چت‌بات
├── services/
│   ├── grading_service.py         هماهنگ‌کننده فرآیند تصحیح
│   ├── openai_grade_engine.py     موتور تصحیح با GPT-4o
│   ├── answer_key_processor.py    پردازش کلید پاسخ با Vision API
│   ├── student_answer_processor.py پردازش برگه دانش‌آموز با Vision API
│   ├── pdf_report_generator.py    تولید گزارش PDF با پشتیبانی فارسی
│   ├── rag_service.py             جستجوی برداری TF-IDF و تولید پاسخ
│   ├── podcast_service.py         تولید اسکریپت و TTS دو مجری
│   ├── math_solver_service.py     حل مسئله متنی و تصویری
│   ├── persian_chatbot_service.py  دستیار مکالمه با حافظه نشست
│   └── storage_manager.py         مدیریت فایل‌های ذخیره‌شده
├── renderer/
│   ├── index.html                 پوسته برنامه و تمام CSS
│   └── app.js                     منطق UI به‌صورت Vanilla JS
└── storage/                       پایگاه داده محلی (JSON)
```

**لایه Electron:** پردازش اصلی در `main.js` هنگام راه‌اندازی یک محیط Python را شناسایی می‌کند (virtualenv، Python Launcher در ویندوز، یا Python سیستمی) و Flask را در پس‌زمینه اجرا می‌کند. یک splash screen نمایش داده می‌شود تا زمانی که endpoint بهداشت `/health` پاسخ مثبت بدهد. ارتباط بین renderer و main process از طریق IPC امن با `contextIsolation: true` انجام می‌شود.

**لایه Flask:** هشت دسته endpoint REST با Blueprint جداگانه سازماندهی شده‌اند. CORS برای localhost پیکربندی شده و حداکثر حجم فایل آپلود ۱۶ مگابایت است.

**پشتیبانی فارسی در PDF:** گزارش‌های تصحیح با `reportlab` تولید می‌شوند و از فونت Noto Sans Arabic به‌همراه کتابخانه‌های `arabic-reshaper` و `python-bidi` برای رندرینگ صحیح متن راست‌به‌چپ استفاده می‌کنند.

## پشته فناوری

| لایه | فناوری |
|---|---|
| محیط دسکتاپ | Electron 35 |

| بسته‌بندی | electron-builder (NSIS/DMG/AppImage) |

## نصب و راه‌اندازی

### پیش‌نیازها

- Node.js نسخه ۱۸ یا بالاتر
- Python 3.10 یا بالاتر
- یک کلید API از GapGPT یا OpenAI

### نصب

```bash
# کلون کردن مخزن
git clone https://github.com/your-org/dahoosh.git
cd dahoosh-electron

# نصب وابستگی‌های Node
npm install

# نصب وابستگی‌های Python
python3 -m venv venv
source venv/bin/activate        # ویندوز: venv\Scripts\activate
pip install -r requirements.txt
```

### پیکربندی

فایل `.env` را ویرایش کنید:

```env
GAPGPT_API_KEY=کلید_API_شما
GAPGPT_BASE_URL=https://api.gapgpt.app/v1
OPENAI_MODEL=gpt-4o
FLASK_PORT=5000
```

### اجرا

```bash
npm start
```

برنامه Flask را روی پورت ۵۰۰۰ راه‌اندازی می‌کند، وضعیت بک‌اند را بررسی می‌کند و پنجره اصلی را نمایش می‌دهد.

### ساخت نصب‌کننده قابل توزیع

```bash
npm run build
```

خروجی در پوشه `dist/` قرار می‌گیرد:
- ویندوز: نصب‌کننده `.exe` با NSIS
- macOS: فایل `.dmg`
- لینوکس: فایل `.AppImage`

## جریان داده در ماژول تصحیح

```
معلم → آپلود کلید پاسخ (PDF/PNG/JPG)
         ↓
    Vision API → JSON ساختاریافته کلید پاسخ
         ↓
معلم → آپلود برگه دانش‌آموز
         ↓
    Vision API → JSON ساختاریافته پاسخ‌های دانش‌آموز
         ↓
    OpenAI GPT-4o → ارزیابی سوال‌به‌سوال با بازخورد فارسی
         ↓
    ReportLab + Noto Sans Arabic → گزارش PDF نهایی
         ↓
معلم → دانلود گزارش
```

## قابلیت‌های ویژه پشتیبانی زبان فارسی

داهوش از روز اول برای فارسی طراحی شده است، نه اینکه بعداً پشتیبانی فارسی به آن اضافه شده باشد:

موتور تصحیح مترادف‌های فارسی را می‌پذیرد و تفاوت‌های نگارشی را در ارزیابی لحاظ می‌کند. RAG با توکنایزر ساده‌ای که هم فارسی هم انگلیسی را پوشش می‌دهد کار می‌کند. گزارش‌های PDF با فونت Noto Sans Arabic و کتابخانه python-bidi رندر راست‌به‌چپ صحیح را تضمین می‌کنند. چت‌بات به‌صورت پیش‌فرض به فارسی پاسخ می‌دهد و در پاسخ‌های درسی به برنامه درسی وزارت آموزش و پرورش ارجاع می‌دهد. حتی بر روی ویندوز، کدبندی UTF-8 در stdout/stderr به‌صورت صریح اعمال می‌شود تا متن فارسی در کنسول بدون خطا نمایش داده شود.

## نمونه داده‌های ذخیره‌شده

مخزن شامل نمونه‌های واقعی ذخیره‌شده در `storage/` است که می‌توان برای آزمایش بلافاصله از آن‌ها استفاده کرد:

- دو کلید پاسخ کامل آزمون با ساختار JSON چند‌سوالی
- سه نتیجه تصحیح واقعی با امتیاز، بازخورد و گزارش کامل هر سوال
